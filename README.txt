MASSI GYM PWA

Contenuto della cartella:
- index.html
- app.webmanifest
- sw.js
- icons/

Per pubblicarla:
1. Carica l'intera cartella su un hosting statico HTTPS.
2. Assicurati che index.html sia nella root del sito.
3. Apri il link da Android con Chrome.
4. In Chrome tocca menu > Installa app / Aggiungi a schermata Home.

Nota:
- Da file locale non si installa come PWA.
- Serve HTTPS oppure localhost.
